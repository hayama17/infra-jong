# インフラ雀 ルール

## 基本構成

- **総牌数**: 36枚
- **プレイヤー数**: 2〜3人
- **初期手札**: 5枚
- **勝利条件**: 3文字の技術用語を2セット完成（計6枚）。最後の1枚はツモまたはロンで揃える。

## ゲームの進行（開発スプリント）

1. **プロビジョニング（配牌）**: 各プレイヤーに5枚ずつ配布
2. **スプリント（ターン）**:
   - **ドロー（ツモ）**: 山札から1枚引く
   - **デプロイ（打牌）**: 不要な1枚を場に捨てる
3. **割り込みアクション**:
   - **ホットフィックス（ポン）**: 他人の捨て牌で用語（3文字）が完成する場合に宣言・公開固定
   - **マージ（ロン）**: 他人の捨て牌で2セット目が完成しあがる
   - **コミット（ツモあがり）**: 自分で引いた牌で完成する

## 牌のデザイン

- **フォント**: JetBrains Mono / Fira Code などの等幅フォント
- **色分け**:
  - Backbone (S, I, A): オレンジ・シアン
  - Finisher (C, P, D...): グレー・ネイビー
