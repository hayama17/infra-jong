# CNK雀 ルール

## 基本構成

- **総牌数**: 48枚
- **プレイヤー数**: 2〜4人
- **初期手札**: 5枚
- **勝利条件**: 3文字の技術用語を2つ完成させる

## 有効用語（16語）

| ファミリー | 用語 |
| --- | --- |
| 最強コア | SRE / DNS / RPC / CRD / SDN |
| プラットフォーム | SDK / CSP / RKE |
| 組織・ロール | CRE / DRE / NRE |
| セキュリティ | CSR / PKE |
| コミュニティ | CNK / CND / PEK |

## ゲームの進行

### 1. プロビジョニング（配牌）
各プレイヤーに5枚ずつ配布してゲーム開始。

### 2. スプリント（ターン）
手番プレイヤーは以下を順に行う。

1. **ツモ（ドロー）**: 山札から1枚引く → 手牌が6枚になる
2. **打牌（デプロイ）**: 手牌から1枚捨てる → 手牌が5枚に戻る

### 3. 割り込みアクション
打牌後、**5秒間**他のプレイヤーが以下を宣言できる。

| アクション | 条件 | 結果 |
| --- | --- | --- |
| **ホットフィックス（ポン）** | 捨て牌＋手牌2枚で用語が完成する | その3枚を公開固定し、手牌から打牌して続行 |
| **マージ（ロン）** | 捨て牌で2つ目の用語が完成しあがる | 即勝利 |

5秒以内に宣言がなければ次のプレイヤーのターンへ。

### 4. コミット（ツモあがり）
ツモった牌で2つ目の用語が完成した場合、打牌の代わりに **コミット** を宣言して勝利。

## 手牌の状態

- 通常: **5枚**（ツモ前）/ **6枚**（ツモ後）
- ホットフィックス後: 3枚が公開固定 ＋ 残り手牌（2〜3枚）

## 終了条件

| 条件 | 結果 |
| --- | --- |
| コミット（ツモあがり）宣言 | 宣言プレイヤーの勝利 |
| マージ（ロン）宣言 | 宣言プレイヤーの勝利 |
| 山札が尽きてツモできない | 引き分け |

## 牌のデザイン

- **フォント**: JetBrains Mono
- **色分け**:
  - Backbone（R / E / C）: オレンジ・シアン — 多くの用語に登場する高頻度文字
  - Middle（S / D / K / N）: グレー
  - Finisher（P）: ネイビー — 少数用語専用の低頻度文字
